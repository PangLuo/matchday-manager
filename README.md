# Matchday Manager

A project scaffolded with [Spec Kit](https://github.com/github/spec-kit) for spec-driven development.

## Getting started

This repository uses the Spec Kit workflow. The project constitution lives in
[`.specify/memory/constitution.md`](.specify/memory/constitution.md).

Typical workflow:

1. `/speckit-constitution` — establish project principles
2. `/speckit-specify` — write a feature specification
3. `/speckit-plan` — produce an implementation plan
4. `/speckit-tasks` — break the plan into tasks
5. `/speckit-implement` — build it

## Layout

- `.specify/` — Spec Kit templates, scripts, memory, and workflows
- `.claude/` — Claude Code skills and settings
